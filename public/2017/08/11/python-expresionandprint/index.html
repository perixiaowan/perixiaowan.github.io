高级序列赋值语句

Python 3.0 扩展序列解包

带*号的名称：一个列表赋值给带*号的名称，该列表收集了序列中没有赋值给其他名称的所有项

备注：

  • 带*号的名称总是会向其赋值一个列表
  • 如果没有剩下的内容可以匹配带*号的名称，会赋值一个空的列表
  • 如果有多个带*号的名称，或如果值少了而没有带*号的名称，以及如果带*号的名称自身没有编写到一个列表中，都会出错
  • 使用列表合并（即+）比使用extend方法更慢

变量命名规则

  • 语法

and del global not with
as elif if or yield

命名惯例

  • 单一下划线开头的变量名_X不会被from module import *语句导入
  • 前后有下划线的变量名_X_是系统定义的变量名
  • 以两个下划线开头，但结尾没有两个下划线的变量名__X是类的本地便利
  • 通常，类变量名以一个大写字母开头，模块变量名以小写字母开头
