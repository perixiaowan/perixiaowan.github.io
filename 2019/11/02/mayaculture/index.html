前言

伯利兹玛雅遗址

伯利兹英文：Belize

玛雅玉米神雕像

玉米是玛雅王国最重要的粮食作物。现存于洛杉矶博物馆。

库库尔坎金字塔

- 底座呈正方形，阶梯朝着正北、正南、正东和正西。
- 四周各有91层台阶，台阶和阶梯平台的数目分别代表了一年的天数和月数。
- 52块有雕刻图案的石板象征着玛雅日历中52年为一轮回。

玛雅王室放血浮雕

- 现存于大英博物馆
- 雕刻于700年~750年
- 表明国王与王后宗教上的伙伴关系，正在完成一项攸关本身权利地位的重大仪式

玛雅香炉

该香炉上坐着一个人物，双腿交叉，周围是一些神话生物。

历史进程

1）公元前1500年，玛雅文明开始发展
2）公元前700年，第一次出现玛雅文字
3）公元前400年，美洲大陆上最早的计时工具–玛雅石制阳历第一次出现
4）公元前200年，玛雅文化成为大路上主流文化
5）200年，一次干旱造成玛雅文明出现一次衰落
6）600年，蒂卡尔城成为美洲最强大城市
7）750年，玛雅商务交流开始没落，战争和骚乱频起。
8）800年，玛雅大城开始废弃，权利逐渐转移到如今天的古巴、墨西哥。
9）899年，蒂卡尔城废弃
10）1517年，西班牙人抵达尤卡坦半岛（Yucatan Peninsula）
11）1695年，发现蒂卡尔城遗迹
12）1697年，最后一个玛雅城市—塔亚索倒塌

第一章 丛林深处的文明古国

在洪都拉斯（Honduras）热带丛林中，首次发现玛雅古文明遗址

1.1 丛林中的古城（约公元前3000年）

1.1.1 三个不同区域

  • 南部区域
    • 玉米和豆类是最主要的作物
  • 中部区域
    • 物产最为丰富的区域
  • 北部区域
    • “塞诺特”形成：地表石灰岩容易崩塌，会形成大量暴露的地下水体，即“塞诺特”，天然的水井。
    • 公园9世纪后，原本在中部区域的文化中心逐渐向北方区域转移

      1.1.2 丰富的石灰石资源

  • 修建祭祀用金字塔神殿、大型广场、娱乐和宗教结合的球场等
  • 石灰岩作为建造神庙以及雕刻各种各样的建筑装饰物的主要材料
  • 奇琴伊察（Chichen Itza），古玛雅城市遗址，位于墨西哥东南部尤卡坦半岛，玛雅古国最大、最繁华的城邦
  • 雕刻工具，黑曜石制作。

    1.1.3 相对封闭的尤卡塔半岛

  • 采用最原始的刀耕火种模式

    1.1.4 科普小知识—美洲三大文明

    均由西班牙人破坏
  • 玛雅文明
  • 阿兹特克文明
  • 印加文明

1.2 发现玛雅（17century-19century）

  • 《埃及、阿拉伯半岛皮特拉和圣地旅行记》
  • 英文中，玛雅甚至成了神秘莫测的代名词。
  • 《尤卡坦仿古览胜记》
  • 科潘玛雅遗址
    位于洪都拉斯科潘省西部的科潘瑞纳斯Copan Ruinas，西与Barrio小镇接壤，遗址位于一片人工隔绝保护出来的占地0.05平方公里正方形的茂密森林中，遗址主金字塔在森林东北角
    公元前200多年为玛雅古王国首都，也是当时的科学文化和宗教活动的中心。
  • 基里瓜遗址
  • 帕伦克遗址，位于墨西哥湾沿岸的热带雨林，商业枢纽
  • 《中美洲、恰帕斯和尤卡坦旅途见闻》两卷，1841年6月，斯蒂芬斯
  • 《尤卡坦旅行记闻》1843年，斯蒂芬斯
  • 蒂卡尔城，玛雅文明保存最完整的遗址，最大的商业枢纽
  • 卡拉克穆尔遗迹，出土了精美玉器

    科普小知识—腓尼基人

    腓尼基人（Phoenician）是一个古老民族，生活在今天地中海东岸相当于今天的黎巴嫩和叙利亚沿海一带，被希腊人称为腓尼基人，是西部闪米特人的西北分支，创立了腓尼基字母 ；腓尼基人善于航海与经商，在全盛期曾控制了西地中海的贸易。

1.3 劫后幸存的玛雅文献（被发现于18世纪）

4本手抄本

1）德累斯顿抄本

- 祭司（指宗教中供奉神祇（qí）的神官或为宗教服务的专职人员）在数百年中陆续抄录绘写的复制品
- 存放在德国德勒斯顿图书馆
- 迄今为止最详细的玛雅抄本
- 包含预言、金星运行规律、日食周期表、天神的生活图画等，还讨论了众神的宇宙北京以及历法和仪式。
- 讲述了曾经存在的三个世界的故事
- 许多内容是作为历法来应用，历法中特定的日子都有主宰吉凶的神明，每个神明图像上方都用玛雅文字写着这个神明的名字。

2）巴黎抄本

- 存放在巴黎国际图书馆
- 主要记述玛雅的预言和天文等内容

3）马德里抄本

- 包含天文、历法、数学、建筑等

4）格罗里那抄本

- 现存放于墨西哥的博物馆
- 主要记载英雄或天神，每页都有一个数字，不知含义。

科普小知识—玛雅纸

用当地无花果树的树心嫩皮制成。将含有大量纤维的树皮捣碎成浆，再加入一种树胶，胶状树浆被摊在平整的石板上，后用木板压制，晒干后，在纸上涂上石灰。

1.4 奥尔梅克文明（公元前1200-公元前100年）

  • 奥尔梅克巨石石像
  • 美洲文明的初曙
  • 代表奥尔梅克文明发展进程的三处文化遗迹
    • 圣洛伦佐文化遗址
    • 拉文塔文化
    • 特雷斯·萨波特斯文化
  • 奥尔梅克发祥地
    • 维拉克鲁斯州
    • 塔巴斯科州
  • 可能是玛雅文明的母体
  • 最让人震撼的是建筑艺术

1.5 发达的玛雅农业（公元3世纪-公元9世纪）

  • 主要是玉米，其次是“拉蒙树”（又称面包树）的面包果。

1.6 玛雅的城市和贸易（公元前1000年）

  • 玛雅最早的城市大约出现在公元前1000年
  • 规模庞大的仪式中心是城市建筑的主体
  • 玛雅后古典时期（公元900年后）的城市建设多带有防御性质的围墙。

1.7 独特的货币可可豆（约公元前2500年开始）

  • 最初的巧克力，称为“诸神之美食”。可可做原料，加上辣椒等搅合在一起做成苦湿辛辣的汁液

科普小知识—苯乙胺

一种生物碱与单胺类神经递质，提升细胞外液中多巴胺的水平，同时抑制多巴胺神经活化（neuron firing），治疗抑郁症

1.8 独特的数学、天文与历法（公元前3000年）

  • 20进位制，从上（大）到下（小）的顺序
  • 三种主要的历法：神历、太阳历和长纪年历
  • 太阴历：即金星历，是指金星环绕太阳一周所用的时间
  • 春分是带来雨季的羽蛇神降临之际，而秋分则是羽蛇神归天而去的时候
  • 奇琴伊察遗址的“螺旋塔”是玛雅人可能将建筑本身用作天文台的一个著名例子

1.9 从自然崇拜到多神崇拜（公元前2世纪~17世纪）

  • 太阳神：造人的天神，创世神的儿子。月亮女神：太阳神的妻子。
  • 保护之神：包含玉米神、雨神等
  • 破坏之神
  • 巴布卡神：共有四个，是养蜂业的保护神
  • 羽蛇神：后古典时期，超越了其他一切神的地位，因为人们对雨水的渴望越来越强烈。

1.10

  • 发生战争是，总指挥被称为‘纳康’
  • 四个阶层：贵族、祭司、平民和奴隶
  • 祭司，被称为‘Ahkin’，太阳之子，天的使臣，神的族裔
  • 祭司分为三等：祭司长（大祭司），高级祭司，普通祭司。还有青年祭司。
  • 玛雅祭司堪称是掌握玛雅文明命脉的人，他们手里有玛雅人走向文明巅峰的金钥匙。

1.11 失落之城—亚特兰蒂斯Atlantis

  • 最先被古希腊柏拉图提出，神秘的国度
  • 猜测希腊的克里特岛是消失的古文明亚特兰蒂斯所在地

2 玛雅人的日常生活

  • 玛雅人不仅把金字塔当做祭祀活动场所，有的还会被用作陵墓。
  • 玛雅人的天堂在13层天之上。
  • 能够一步登天的人：自杀者，战死的武士，做人祭牺牲的人，难产而死的女人，祭司
  • 玛雅人的名字由祭司来取
  • 玛雅人举行的人祭场面，最中心的程序是剜心。
  • 头部越长越高贵；头部扁长为美。斗鸡眼也是高贵的象征
  • 身披美洲豹皮的都是玛雅的精英武士
  • 玛雅男人标准穿戴：遮羞布，披肩，凉鞋和头饰
  • 涂面纹身
    • 未婚男人-黑色
    • 已婚男人-红色
    • 祭司-蓝色
    • 祭品：黑白条纹
    • 勇敢的武士：红黑
