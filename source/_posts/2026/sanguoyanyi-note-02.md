# 第二回 张翼德怒鞭督邮 何国舅谋诛宦竖
## 1. 引子
第二章继续讲述了东汉末年的社会背景和主要人物的发展。通过对一些重要事件的描绘，如黄巾起义、董卓入京等，进一步展示了当时的政治动荡和社会不安。同时，第二章也介绍了一些新的重要人物，如刘备、关羽、张飞等，为后续的故事发展做好了铺垫。
## 2. 主要人物介绍
- 严政：黄巾起义贼将。刺杀张宝，献首投降。
- 赵弘、韩忠、孙仲：黄巾余党。
- 孙坚：孙武子之后。
- 刘恢：刘备的叔父，汉景帝的儿子，汉武帝的孙子，汉昭帝的曾孙，汉宣帝的玄孙。被封为中山王。
- 刘陶：东汉忠臣，谏议大夫，以直言敢谏、怒斥十常侍而留名，最终被宦官陷害致死。
- 陈耽：东汉司徒，撞阶死谏，与刘陶同殉汉室，是汉末最悲壮的三公忠臣。
- 刘虞 = 汉室忠臣、幽州仁牧、刘备的贵人；宁死不称帝，最终死于军阀公孙瓒之手。
- 蹇硕 = 东汉唯一掌军宦官、灵帝托孤之人，欲杀何进立刘协而败，被十常侍出卖，是东汉宫廷权力斗争的第一个牺牲品。
- 潘隐 = 蹇硕麾下司马，临阵倒戈救何进，以一己之力促成汉末大乱，是个没立场、只保命的关键小人物。
- 袁绍 = 汉末第一豪门、讨董盟主、河北霸主；兵最多、地最广，却因优柔寡断在官渡输给曹操，一手好牌打烂。曹操评价“袁绍色厉胆薄，好谋无断；干大事而惜身，见小利而忘命。”
- 东汉末年著名文学家，建安七子之一。在《三国演义》中，他是因一篇骂遍曹操祖宗三代的檄文而名留千古的 “笔刀” 谋士。
## 3. 文章解读
- 突槊 = 突然用长矛攻击。
- 除授 = 朝廷正式下旨，任命某人担任某官职
- “此必破黄巾有功者，不得除授，故生怨言。权且教省家铨注微名，待后却再理会未晚。”
  - 这些人一定是平定黄巾立了功，却没被封官授职，所以才产生怨言。暂且让吏部的人给他们登记个小功名，等以后再慢慢处置他们也不晚。
  - 省家：指朝廷吏部、办事衙门
  - 铨注：登记、考核、记名
  - 微名：低微的功名、小头衔
- “惟玄德以鞭指回答”
  - 只有刘备用马鞭指着（对方）来回答 / 用马鞭示意作答
- 沙汰 = 清理、淘汰、裁员、整顿。
- 督邮（dū yóu），是汉代至魏晋时期，郡太守手下最核心的监察官，相当于郡里的 “纪委书记”+ 特派巡视员。
- 吾思枳棘丛中，非栖鸾凤之所 = 我这样的人，不能长久待在这种污浊、憋屈的小地方，跟这帮小人混在一起。
## 4. 诗句
- “欲除君侧宵人乱，先扫十常侍门户尘。”
  - 想要清除君主身边的宵小之人，首先要把十常侍这个大祸害清理干净。
